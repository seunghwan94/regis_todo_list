# Regis Todo List
**Reg**ular **I**nspection **S**ystem (정기 점검 Todo List 프로그램)

<img width="1914" height="908" alt="image" src="https://github.com/user-attachments/assets/fc276c72-b604-4584-beac-57bbce7844c4" />

정기 점검 업무를 체계적으로 관리할 수 있도록 만든 웹 기반 관리 도구입니다.
고객사별 정기 점검 스케줄을 자동 정리하고, 체크리스트 기반으로 진행 상황을 손쉽게 확인할 수 있습니다.

> 본 프로그램은 Docker + Python + SQLite 기반으로 누구나 간단히 설치 후 사용할 수 있습니다.

### 주요 기능
#### 고객사 & 상세 고객사 관리
- 고객사 등록
- 상세 고객사(홈페이지, 다이렉트, VDI 등) 추가 가능

#### 정기 점검 생성
- 점검 유형(사내점검 / 방문점검)
- 서명 방식(메일 / 방문)
- 스케줄 선택
    - 매월
    - 분기 (예: 1,4,7,10 / 3,6,9,12 …)
    - 직접 커스텀 입력
- 담당자 정보 입력 (이름 / 연락처 / 이메일)

#### 3. 체크리스트 기능

- 항목 설명 입력
- 파일 첨부 가능
- 완료/미완료 상태 월별로 저장
- 체크는 월 단위로 독립 관리됨
(예: 2025년 12월 완료해도 2026년 12월은 새로 진행됨)

#### 4. 대시보드
- 연도/월 단위 점검 현황 표시
- 진행 중 / 완료 구분
- 고객사 필터
- 다음 달 예정 확인 가능
- 월별 진행 현황 뱃지 표기 (예: 2/5)

#### 5. 전체 리스트
- 수행되는 점검만 표시하거나 숨길 수 있음
- 체크하면 한 번에 숨김/보임 처리 가능
- 정기점검 수정 & 삭제 가능
- 체크리스트도 수정/추가/삭제 가능

#### 6. 첨부파일 다운로드
- 각 항목에 첨부한 파일은 클릭하여 다운로드 가능

### 설치 및 실행 방법
1) 프로젝트 다운로드

2) Docker 이미지 빌드

```
docker build -t todo_app .
```

3) 실행

```
docker run -d --name todo_app_container -p 8000:8000 todo_app
```

4) 접속

브라우저에서 아래 주소로 접속
```
http://localhost:8000
```

### 📂 데이터 저장

모든 데이터는 database.db에 저장됩니다.

별도의 DB 설치 없이 바로 사용 가능
